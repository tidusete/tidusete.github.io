Create a Voteweb Bot with Python part 1

Hi buddies, in this post we will talk about how to make a simple vote bot with python using the famous Selenium library.

Selenium is a suite for automating web applications for testing purposes. But on the naughty side, it can help us to automate all the painful voteweb-based tasks.

Our first step, it will be having a stable environment and install selenium library

  pip install selenium

Once we have finished installing the package, we will have to proceed to install the drivers. As Selenium requires a Driver to interface with the chosen browser, we will have to go to https://selenium-python.readthedocs.io/installation.html#drivers and pick one. In the example below, we will use chromeDriver.

We will have to choose the Chromedriver accord to our version of chrome. To know which version we have installed on chrome, we just have to go to (…) -> About Chromium Ensure to place the binary on your binary PATH e.g. /usr/bin or /usr/locate/bin

After install the selenium package and the chromeDriver’s drive, we will start looking for the target’s webpage.

Roadmap: { Accept Cookies -> Login -> Go To Vote }

On the home webpage, we can identify 2 important items. The cookies and the User Area.

To let the script accept the Cookies:

from selenium.webdriver.support.ui import WebDriverWait
# Define a wait that will help us to not execute the code before we can identify an element which class name is cookieinfo-close
wait = WebDriverWait(driver, 10) 
wait.until(EC.element_to_be_clickable((By.CLASS_NAME,"cookieinfo-close")))
#Click the button to accept cookies
driver.find_element_by_class_name("cookieinfo-close").click()

To let the script to login as a User:

driver.find_element_by_name("login_username").send_keys("FAKE_USER")
driver.find_element_by_name("login_password").send_keys("FAKE_PASSWD")  
driver.find_element_by_name("login_submit").send_keys(Keys.RETURN)

After login as a user, the new page will be loaded and we will force it to go to the vote option:

#Wait until the webpage has been loaded the ucp_buttons
wait = WebDriverWait(driver, 10) 
wait.until(EC.element_to_be_clickable((By.ID, 'ucp_buttons')))
driver.get('https://fake.com/vote')  #gotovote

And finally, the part of the script which will click all the buttons will be:

wait = WebDriverWait(driver, 10)
wait.until(EC.element_to_be_clickable((By.ID, 'vote')))
#Click first button 
driver.find_element_by_xpath('//*[@id="vote"]/tbody/tr[2]/td[3]/form/input[2]').click()
pause.sleep(1)

Is interesting to take a look on the python documentation about Selenium.

The URL is https://selenium-python.readthedocs.io/locating-elements.html

Also, is really important to know how we can easily identify this properties using the Development tools kit from chrome and use sometimes the Xpath identifier.

Finally, once we understand all the parts of the script, it’s time to join it all and build the bot.

from selenium import webdriver
from selenium.webdriver.common.keys import Keys
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
import time
 
pause = time
driver = webdriver.Chrome()
driver.get("https://fakeweb.com/")

#Cookies
wait = WebDriverWait(driver, 10)
wait.until(EC.element_to_be_clickable((By.CLASS_NAME,"cookieinfo-close")))
driver.find_element_by_class_name("cookieinfo-close").click()

#Login
driver.find_element_by_name("login_username").send_keys("fake")
driver.find_element_by_name("login_password").send_keys("fakepass")  
driver.find_element_by_name("login_submit").send_keys(Keys.RETURN)

#Wait page for loading
wait = WebDriverWait(driver, 10)
wait.until(EC.element_to_be_clickable((By.ID, 'ucp_buttons')))
#gotovote
driver.get('https://fake.com/vote')

#Wait page for loading
wait = WebDriverWait(driver, 10)
wait.until(EC.element_to_be_clickable((By.ID, 'vote')))

#button1
driver.find_element_by_xpath('//*[@id="vote"]/tbody/tr[2]/td[3]/form/input[2]').click()
pause.sleep(1)

#CloseProgram
driver.close()

At this point, we have a script which will vote every time you invoke it.
